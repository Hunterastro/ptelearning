## 📋 **用户文件格式要求**

### **Excel文件格式**
```
第1列: Collocation（搭配词）
第2列: Sentence（例句）
第3列: Meaning（英文释义）
第4列: 中文翻译
```

### **系统字段映射**
- `Collocation` → `phrase`（学习短语）
- `Sentence` → `example`（例句）
- `Meaning` → `translation`（释义说明）
- `中文翻译` → `meaning`（中文释义）
